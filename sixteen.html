10 Write a PL/SQL block to declare a constant value for PI and calculate area of a circle. (Constants)
Show Answer

22 Write a PL/SQL block that uses all concepts: variable, constant, datatype, and output. (Full Practice)

DECLARE -- Constant declaration c_company CONSTANT VARCHAR2(30) := 'Tech Solutions'; -- Variable using %TYPE datatype v_emp_name EMP.ENAME%TYPE; -- Standard variable v_project_count NUMBER := 5; BEGIN -- Data Retrieval SELECT ENAME INTO v_emp_name FROM EMP WHERE EMPNO = 7839; -- Output combined results DBMS_OUTPUT.PUT_LINE('Company: ' || c_company); DBMS_OUTPUT.PUT_LINE('Employee: ' || v_emp_name); DBMS_OUTPUT.PUT_LINE('Assigned Projects: ' || v_project_count); END;
