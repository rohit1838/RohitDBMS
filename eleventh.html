1 What is the purpose of the NVL function in SQL?
Show Answer
The **NVL** function is used to replace a NULL value with a specific value. **Syntax:** NVL(expression, replacement_value) If the expression is NULL, it returns the replacement value; otherwise, it returns the original value.
2 Display employee name and commission. If commission is NULL, display 0. EMP (ENAME, COMM)
Show Answer
SELECT ENAME, NVL(COMM, 0) AS COMMISSION FROM EMP;
3 What is the difference between NVL and NVL2?
Show Answer
- **NVL(exp, val1):** Returns val1 if exp is NULL. - **NVL2(exp, val1, val2):** Returns val1 if exp is NOT NULL, and returns val2 if exp is NULL.
4 Display employee name and a column named 'Income_Type'. If commission is NULL, display 'Salary Only', otherwise display 'Salary + Comm'.

5 Explain the NULLIF function.
Show Answer
The **NULLIF** function compares two expressions. If they are equal, the function returns NULL. If they are not equal, it returns the first expression. **Syntax:** NULLIF(exp1, exp2)

7 What is the COALESCE function?
Show Answer
**COALESCE** returns the first non-null expression in a list. It can take multiple arguments and evaluates them in order. **Syntax:** COALESCE(val1, val2, ..., valN)
