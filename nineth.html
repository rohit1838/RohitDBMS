1 What is a subquery in SQL?
Show Answer
A subquery is a query within another query. The outer query is called the main query, and the inner query is the subquery. The subquery executes first and its result is used by the main query.
2 What are the different types of subqueries?
Show Answer
1. Single-row subquery: Returns only one row. 2. Multi-row subquery: Returns multiple rows. 3. Multiple column subquery: Returns multiple columns. 4. Correlated subquery: Reference columns from the outer query.
3 What is the difference between a subquery and a join?
Show Answer
- Subquery: Used to return data that will be used in the main query as a condition. Usually easier to read. - Join: Used to combine rows from two or more tables based on a related column. Joins are generally faster in performance for large datasets.
4 What is a single-row subquery?
Show Answer
A subquery that returns exactly one row of results. It uses single-row operators like =, >, <, >=, <=, or <>.
5 What is a multi-row subquery?
Show Answer
A subquery that returns one or more rows. It uses multi-row operators like IN, ANY, or ALL.
6 Display employees whose salary is greater than the average salary. EMP (EMPNO, ENAME, JOB, MGR, HIREDATE, SAL, COMM, DEPTNO)
Show Answer
SELECT * FROM EMP WHERE SAL > (SELECT AVG(SAL) FROM EMP);

14 Display employees whose salary is greater than company average but less than department average. EMP (EMPNO, ENAME, JOB, MGR, HIREDATE, SAL, COMM, DEPTNO)
Show Answer
SELECT * FROM EMP e WHERE SAL > (SELECT AVG(SAL) FROM EMP) AND SAL < (SELECT AVG(SAL) FROM EMP WHERE DEPTNO = e.DEPTNO);
15 Display employees earning top 3 salaries in each department. EMP (EMPNO, ENAME, JOB, MGR, HIREDATE, SAL, COMM, DEPTNO)
Show Answer
SELECT * FROM EMP e1 WHERE 3 > (SELECT COUNT(DISTINCT SAL) FROM EMP e2 WHERE e2.DEPTNO = e1.DEPTNO AND e2.SAL > e1.SAL);
16 Display nth highest salary using subquery. Generic Formula
Show Answer
SELECT DISTINCT SAL FROM EMP e1 WHERE (N-1) = (SELECT COUNT(DISTINCT SAL) FROM EMP e2 WHERE e2.SAL > e1.SAL); -- Note: Replace 'N' with the desired number (e.g., 5 for 5th highest).

24 Display employees whose salary is less than ANY employee in department 10. Part 1
